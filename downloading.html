Downloading files using <a> tags

When an HTML anchor tag has the download attribute, it will download the target instead of navigating to it.

So this code snippet:

... will look like this:

click here

...and when it's clicked, the file will be downloaded

  • You can specify a file name. If you don't, the original file's name will be used
  • The downloaded rescource needs to be from the same origin as the initiator.
  • Files can only be downloaded to your downloads folder.
